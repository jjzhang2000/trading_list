# 股票分析项目

## 项目概述

本项目是一个基于历史数据的股票分析系统，主要用于分析上证A股股票并进行排名，帮助投资者发现潜在的投资机会。项目从网络数据源获取历史数据，并结合技术指标进行分析和评分。

## 项目结构

```
trading_list/
├── stock_analyzer.py      # 股票分析主脚本
├── config.py             # 配置文件
├── screen_stocks.py      # 股票筛选脚本
├── stock_strategy.py     # 股票策略脚本
├── requirements.txt      # 依赖项文件
├── .env                  # 环境变量文件
├── .gitignore            # Git忽略文件
└── data/
    ├── extract_data.py    # 前复权价格数据获取脚本
    ├── init_db.py         # 数据库初始化脚本
    └── stock_data.db         # 股票数据库
```

## 文件功能说明

### 1. stock_analyzer.py

**功能**：股票分析主脚本，负责计算技术指标和对股票进行评分排名。

- **技术指标计算**：
  - 移动平均线 (MA)
  - 相对强弱指标 (RSI)
  - 移动平均收敛发散 (MACD)
  - 其他技术指标

- **股票评分系统**：
  - 基于技术指标的综合评分
  - 股票排名功能
  - 生成投资建议

### 2. config.py

**功能**：项目配置文件，存储各种配置参数。

- **配置项**：
  - 数据获取参数
  - 技术指标计算参数
  - 评分系统参数
  - 其他项目配置

### 3. screen_stocks.py

**功能**：股票筛选脚本，根据特定条件筛选股票。

- **筛选条件**：
  - 基于技术指标的筛选
  - 基于基本面的筛选
  - 自定义筛选策略

### 4. stock_strategy.py

**功能**：股票策略脚本，实现各种交易策略。

- **策略实现**：
  - 趋势跟踪策略
  - 均值回归策略
  - 动量策略

### 5. extract_data.py (位于 data/ 目录)

**功能**：从网络数据源获取前复权价格数据并存储到数据库。

- **数据获取**：
  - 从东方财富API获取前复权价格数据
  - 从akshare获取前复权价格数据
  - 支持增量更新，只获取最新数据

- **复权因子检测**：
  - 检测复权因子变动
  - 当复权因子变动时重新下载所有数据

- **数据存储**：
  - 存储前复权价格数据到SQLite数据库
  - 记录股票基本信息和数据时间范围

### 6. init_db.py (位于 data/ 目录)

**功能**：初始化数据库，清除历史数据或创建新数据库。

- **数据库检查**：
  - 检查数据库是否存在
  - 如果存在，清除其中的所有历史数据
  - 如果不存在，创建新数据库

- **表结构创建**：
  - 创建股票日线数据表（stock_daily）
  - 创建股票信息表（stock_info）
  - 创建索引以提高查询效率

## 依赖项

项目依赖以下Python库：

- pandas
- numpy
- akshare

可以通过以下命令安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

1. **初始化数据库**：
   ```bash
   python data/init_db.py
   ```

2. **提取历史数据**：
   ```bash
   python data/extract_data.py
   ```

3. **分析股票**：
   ```bash
   python stock_analyzer.py
   ```

4. **筛选股票**：
   ```bash
   python screen_stocks.py
   ```

## 数据来源

- **历史数据**：通过 akshare 从东方财富网获取
- **复权数据**：通过东方财富API和akshare获取前复权价格数据

## 注意事项

1. **数据完整性**：
   - 系统会自动处理API返回的数据格式异常
   - 确保网络连接正常，以便获取最新的股票数据

2. **API限制**：
   - 系统会添加适当的延迟，避免触发API频率限制
   - 若遇到API限制，程序会自动尝试使用备用数据源
